Library Management System (Java Swing + MySQL)
----------------------------------------------

What's included:
- src/DBConnection.java
- src/Book.java
- src/Library.java
- src/LibraryGUI.java
- database/library_db.sql
- README.txt (this file)

Quick setup:
1. Install Java (JDK 8 or higher) and MySQL Server.
2. Create the database:
   - Open MySQL client and run the SQL file: database/library_db.sql
     (This creates database 'library_db' and table 'books'.)
3. Update DBConnection.java if your MySQL username/password are different:
   - By default USER='root' and PASSWORD=''. Change accordingly.
4. Compile and run:
   - From the src folder:
     javac *.java
     java LibraryGUI
Notes:
- You need the MySQL JDBC driver (Connector/J) on the classpath.
  Download the driver and add the JAR when running, e.g.:
  javac *.java
  java -cp .:mysql-connector-java-8.0.XX.jar LibraryGUI   (Linux/Mac)
  java -cp .;mysql-connector-java-8.0.XX.jar LibraryGUI   (Windows)
- If Java complains about timezone, DB URL already sets serverTimezone=UTC.
- If you want a console-only version, replace LibraryGUI.main with a text menu.
